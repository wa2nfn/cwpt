cwpt2 - c.w. practice text (a Morse Code learning tool)

If you're learning Morse Code or working to increase your proficiency using a Morse tutor/trainer software program, then you may benefit from cwpt2's features.

This free tool does NOT replace the need for your Morse tutor software, it supplements their practice features. Cwpt2 creates practice material in many formats that your current tutor software may not. Through its many options (over 45), you build practice sessions that support your learning needs.

cwpt2 is a command line tool, its only 1 file so there is no complex install needed to try it 

Download it, give it execute permissions for you environment.
To run from the command line on Windows, hold the "windows" key and press "R", click "OK" on the pop-up window, cd to the directory where you placed the cwpt2.exe file,
execute with ".\cwpt2.exe etc."

MP3 demo links below: 

 Each word plays at a different speed.
i.e. |w20 women |w35 young |w30 above |w20 darcy |w25 after |w30 party |w40 large |w35 which |w25 being 

 A word is played several times, each time at an increased speed, then the speed
returns to the lowest rate and a new word begins.
i.e. |w20 which |w23 which |w26 which |w20 after |w23 after |w26 after |w20 being 

 A few words are played at the low speed, then a shorter set is played at incremental speed increase; the cycle continues until all the words are sent (thus only 2 speed rates)
i.e. |w20 every noble darcy party |w25 hurst about |w20 could great women young |w25 which above |w20 after being 

 The entire practice session is divided into equal length sections, starting at the lowest speed, then each section is sent a an incrementally faster speed than the last (so speed is an ever increasing ramp up).
i.e. |w20 noble above large |w25 about great darcy |w30 which party young |w35 women after every 

 This example inserts a code group, after every four text words in addition to speed changes.
i.e. |w20 which hurst great /,-?- every |w25 about young .a.== |w20 being could above -0/.a party |w25 

 This example stretches INTER word spacing - 2.5 times the normal for this 20 wpm speed.
i.e. |W2.5 |w20 more spacing between words 

 This example plays plain text at a different speed than code group
i.e.  |w20 two one one |w13 szw4?4 |w20 ten ten nine |w13 wlyt36 |w20 two nine four
